
* Title:            Package for the hbus bus protocol.
* Name:             hbus
* Version:          0.1 
* Description:
 
This is the hbus OVC.  The hbus is a basic read/write bus protocol
implemented in SystemVerilog, designed to work with the open source
UVM library 1.0ea

* Directory structure:

This package contains the following directories: 

  sv/ - All SystemVerilog source files
  tb/ - Verification Environment example        

Note: The hbus_pkg.sv and hbus.svh files present in the references folder contains the original structure of files. You will not find these files in the hbus folder. To encript the files, few components codes are directly written in the hbus_pkg.sv which is not recommended. Please follow the structure shown in the references folder.
